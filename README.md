# 사회적경제 퀴즈 게임

행사장 와이파이 안에서 노트북/PC를 서버로 띄우고 태블릿 2대가 같은 문제 목록으로 퀴즈를 푸는 웹앱입니다. 인터넷 연결이나 추가 패키지 설치 없이 Node.js만 있으면 실행됩니다.

## 실행

```bash
npm start
```

브라우저에서 아래 주소로 접속합니다.

- 참가자 화면: `http://localhost:3000`
- 관리자 화면: `http://localhost:3000/admin`
- 관리자 기본 비밀번호: `3847`

## 행사장 접속 주소 예시

서버를 실행하면 터미널에 같은 와이파이 접속 예시가 표시됩니다.

예를 들어 노트북 IP가 `192.168.0.25`라면:

- 태블릿 참가자 화면: `http://192.168.0.25:3000`
- 관리자 화면: `http://192.168.0.25:3000/admin`

태블릿과 서버 노트북은 같은 와이파이에 연결되어 있어야 합니다.

## 와이파이가 전혀 없을 때

와이파이가 전혀 없으면 태블릿 2대가 서버와 통신할 수 없으므로, 각 태블릿이 독립적으로 실행되는 오프라인 파일을 사용해야 합니다.

```bash
npm run build:offline
```

생성된 `offline-quiz.html` 파일을 행사 전에 각 태블릿에 넣고 브라우저로 열면 됩니다. 두 태블릿은 같은 문제가 들어간 파일을 각각 실행하고, 점수는 각 태블릿에서 따로 계산됩니다.

주의할 점:

- 오프라인 파일은 서버와 연결되지 않으므로 행사 중 문제 수정이 두 태블릿에 자동 반영되지 않습니다.
- 문제나 디자인을 바꾸면 다시 `npm run build:offline`을 실행하고 새 `offline-quiz.html`을 태블릿마다 넣어야 합니다.
- 관리자 비밀번호 보호는 서버 방식에서만 의미가 있습니다. 완전 오프라인 파일은 참가자 전용으로 쓰는 것이 안전합니다.

## 관리자 비밀번호 변경

실행할 때 비밀번호를 지정할 수 있습니다.

```bash
ADMIN_PASSWORD=원하는비밀번호 npm start
```

## 문제 저장

문제는 서버의 `data/questions.json` 파일에 저장됩니다. 참가자 태블릿은 같은 서버에서 이 목록을 받아오며, 점수는 각 태블릿 브라우저에서 따로 계산됩니다.

참가자 화면은 오답이나 시간초과가 나오면 같은 문제를 다시 보여줍니다. 정답을 맞혀야 다음 문제로 넘어가며, 마지막 문제까지 맞히면 점수표 없이 `MISSION COMPLETE` 화면과 폭죽 효과가 표시됩니다.

## 디자인 설정

관리자 화면의 `디자인 설정`에서 참가자 화면의 시작 작은 문구, 제목, 시작 문구, 시작 배경 사진, 시작 배경 어둡게, 배경색, 버튼 색상, 글자 색상, 정답/오답 색상, 마지막 작은 문구, 마지막 화면 문구, 마지막 배경 사진, 마지막 배경 어둡게, 제한시간, 폭죽 효과 켜기/끄기를 바꿀 수 있습니다.

시작 화면과 마지막 화면의 글자는 사진 위에 직접 놓이지 않고 읽기 쉬운 중앙 패널 안에 표시됩니다. 배경 사진은 패널 주변에 보이며, `배경 어둡게` 값을 높이면 사진이 더 진하고 어둡게 보입니다.

저장한 디자인은 `data/settings.json`에 저장됩니다. 태블릿 참가자 화면은 새로고침하면 저장된 디자인을 다시 불러옵니다.

## 한꺼번에 문제 등록 형식

관리자 화면의 한꺼번에 등록 칸에 아래 형식으로 붙여넣으면 됩니다. 문제 사이에는 빈 줄을 넣어주세요.

```text
OX
문제: 사회적기업은 사회적 목적을 추구하면서 영업활동을 한다.
정답: O

객관식
문제: 협동조합의 기본 원칙과 가장 가까운 것은?
1. 조합원의 민주적 운영
2. 한 사람의 독점 운영
3. 단기 이익만 우선
4. 지역사회 배제
정답: 1
```
