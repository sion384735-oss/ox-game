# 사회적경제 퀴즈 게임

행사장 와이파이 안에서 노트북/PC를 서버로 띄우고 태블릿 2대가 같은 문제 목록으로 퀴즈를 푸는 웹앱입니다. 인터넷 연결이나 추가 패키지 설치 없이 Node.js만 있으면 실행됩니다.

## 실행

```bash
npm start
```

브라우저에서 아래 주소로 접속합니다.

- 참가자 화면: `http://localhost:3000`
- 관리자 화면: `http://localhost:3000/admin`
- 관리자 기본 비밀번호: `3847`

## 행사장 접속 주소 예시

서버를 실행하면 터미널에 같은 와이파이 접속 예시가 표시됩니다.

예를 들어 노트북 IP가 `192.168.0.25`라면:

- 태블릿 참가자 화면: `http://192.168.0.25:3000`
- 관리자 화면: `http://192.168.0.25:3000/admin`

태블릿과 서버 노트북은 같은 와이파이에 연결되어 있어야 합니다.

## 관리자 비밀번호 변경

실행할 때 비밀번호를 지정할 수 있습니다.

```bash
ADMIN_PASSWORD=원하는비밀번호 npm start
```

## 문제 저장

문제는 서버의 `data/questions.json` 파일에 저장됩니다. 참가자 태블릿은 같은 서버에서 이 목록을 받아오며, 점수는 각 태블릿 브라우저에서 따로 계산됩니다.

## 한꺼번에 문제 등록 형식

관리자 화면의 한꺼번에 등록 칸에 아래 형식으로 붙여넣으면 됩니다. 문제 사이에는 빈 줄을 넣어주세요.

```text
OX
문제: 사회적기업은 사회적 목적을 추구하면서 영업활동을 한다.
정답: O

객관식
문제: 협동조합의 기본 원칙과 가장 가까운 것은?
1. 조합원의 민주적 운영
2. 한 사람의 독점 운영
3. 단기 이익만 우선
4. 지역사회 배제
정답: 1
```
